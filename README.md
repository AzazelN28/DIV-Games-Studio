cross platform IDE, compiler and runtime for

* DOS
* Windows
* OSX
* Android 
* Amazon Fire
* Amazon Kindle
* Ouya Console
* Linux
* HTML / Javascript (emscripten - runtime export only)

* PSP
* Nintendo DS
* Nintendo Wii
* XBOX (XNA)
* IOS (iPhone / iPad)
* Windows Mobile

# README #

DIV Games Studio 2, ported, translated and upgraded. 

cross platform IDE, compiler and runtime for

* Windows
* OSX
* Android
* Linux
* HTML / Javascript (emscripten)
* Raspberry Pi


Future Possible Platforms:

* PSP
* Nintendo DS
* GCW-Zero
* Pandora Console
* Nintendo Wii

100% compatible with existing DIV2 code (except mode8!)
